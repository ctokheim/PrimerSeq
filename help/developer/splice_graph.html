splice_graph.py¶

Author: Collin Tokheim

Splice Graph¶

The splice_graph.py deals with gene structure as a (weighted) directed acyclic graph (DAG) normally known as a splice graph. Like you would expect, the SpliceGraph class represents gene structure as a splice graph.

Transcripts Overlapping Target¶

get_from_gtf_using_gene_name() returns the all transcripts from a gene if the gene overlaps the user’s target. get_weakly_connected_tx() returns all transcripts that are weakly connected to the user’s target. This approach is used when gene IDs are not valid and thus can not be used.

Flanking Exons¶

If read counts are used (psi < 1) then splice_graph uses the get_sufficient_psi_exons() function to find flanking exons with at least a user defined inclusion level. If psi == 1 then flanking exons are only determined by the biconnected componenets algorithm using get_flanking_biconnected_exons().

Only try to find weights for already existing edges in the graph. This function is intended to add weight values to edges defined in the gtf annotation.

Create a nx DiGraph from list of tx in gene. FILTER_FACTOR defines a cutoff for using a tx of a gene. A tx must have x num of exon where x > MAX tx exon num / FILTER_FACTOR.

set_graph_as_nodes_only(exons)[source]¶

Simple function that makes a DAG (nx.DiGraph) with only nodes and no edges. Meant to be used to before add_all_possible_edge_weights.

Calculate psi for the target exon for each bam file.

splice_graph.get_flanking_biconnected_exons(name, target, sGraph, genome)[source]¶

Defines flanking exons as exons that cannot be skipped in the graph structure. Theese exons are 100% included and do not need estimation of inclusion level.

splice_graph.get_from_gtf_using_gene_name(gtf, strand, chr, start, end)[source]¶

This function finds the first gene in the gtf that completely contains the target interval. I should really think about checking for multiple genes instead of just returning.

splice_graph.get_sufficient_psi_exons(name, target, sGraph, genome, ID)[source]¶

Utilizes the ExonSeek class to find flanking exons that are good enough to be called “constitutive”.

splice_graph.get_weakly_connected_tx(gtf, strand, chr, start, end, plus_or_minus=1000000)[source]¶

This function is meant to handle tx annotations without gene ids. Currently this is a function outside of the SpliceGraph class but it may be beneficial to later include this as a method.

splice_graph.main(options, args_output='tmp/debug.json')[source]¶

The gtf main function is the function designed to be called from other scripts. It iterates through each target exons and returns the necessary information for primer design.

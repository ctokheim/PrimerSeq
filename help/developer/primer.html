primer.py¶

Author: Collin Tokheim

Description: primer.py can design primers from the command line. Note that PrimerApp.py (the GUI) just uses the primer module to design the primers.

primer.gene_annotation_reader(file_path, FILTER_FACTOR=2)[source]¶

creates two data structures from gtf:

gene_dict is a dictionary with gene_id’s as keys:
  • gene_dict[‘chr’][‘My_favorite_gene’][‘graph’] = list of list of exons
  • gene_dict[‘chr’][‘My_favorite_gene’][‘strand’] = gene strand (+, -)
  • gene_dict[‘chr’][‘My_favorite_gene’][‘chr’] = chromosome
  • gene_dict[‘chr’][‘My_favorite_gene’][‘start’] = start of gene
  • gene_dict[‘chr’][‘My_favorite_gene’][‘end’] = end of gene
  • gene_dict[‘chr’][‘My_favorite_gene’][‘exons’] = the set of exons (nodes)

primer.primer3(options, primer3_options)[source]¶

The primer.py main function uses the gtf module to find information about constitutive flanking exons for the target exons of interest. It then designs primers by calling primer3. Next it parses the primer3 output and outputs the final results to a file. The output file is then emailed to the designated address in the command line parameters.
